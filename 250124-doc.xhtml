오랜만에 lyx를 사용하니 obsidian, dynalist 보다 안정된 느낌이 든다.
lyx의 단점 ( obsidian, dynalist 장점 ) 이 무엇이였지 ?
- 인터넷 공유 : 이것은 html 또는 pdf 로 자주 올리면 될듯, ( but html 은 색, tikz 지원 안됨 )
- 전역 검색 ? - 태그 체계 ( hierarchy ) ? - 외부 문서 링크 관리 ? → 사실 이것들 그렇게 많이 쓰지 않음
lyx의 장점
- 체계 ( hierarchy ) / 표, 수식 / 폰트 품질
obsidian, dynalist 의 단점
- 체계가 없어서 빨리 찾기 불편, tag로 검색하는 것 불편 ( 여러 문서에 퍼져 있어서 자료 존재, 따라서 자료를 한곳에 모아야 편함 )
